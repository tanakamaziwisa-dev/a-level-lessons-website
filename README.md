# Tanaka A-Level Lessons Website

A modern, multi-page A-Level tutoring website built with HTML, CSS, and JavaScript.

## Quick edits

### Change the WhatsApp number
1. Open `assets/js/main.js`.
2. Update the `whatsappNumber` value.

```js
const whatsappNumber = "+2637XXXXXXXX";
```

### Change prices
1. Open `pricing.html`.
2. Update the price placeholders (`$X per hour`) inside each pricing card.

### Publish for free with GitHub Pages
1. Push this repository to GitHub.
2. In your GitHub repo, go to **Settings â†’ Pages**.
3. Under **Source**, select `Deploy from a branch`.
4. Choose the `main` branch and `/root` folder.
5. Save. Your site will appear at the Pages URL provided by GitHub.

## Local usage
Open `index.html` in any browser to view the site.
